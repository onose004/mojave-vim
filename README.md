Mojave for Vim
===

A dark theme for vim

![Mojave in Vim](screenshot/vim_js.png)

## Installation

Put mojave.vim file in your ~/.vim/colors/ directory and add the following line to your ~/.vimrc:

```vim:~/.vimrc
syntax enable
colorscheme mojave
```

### light-line

```vim:~/.vimrc
let g:lightline = {'colorscheme': 'mojave'}
```
